# TerraFlux-Workspace
This folder is the canonical workspace for TerraFlux projects.
- content/: approved notes
- suggestions/: agent suggestions and drafts
- attachments/: media and PDFs
- scripts/: automation
- tools/: validation and helper scripts
- backups/: snapshots
- logs/: runtime logs
- templates/: note templates
- docs/: runbooks and onboarding
